# ⚡ Grid-Sync

AI-Based Microgrid Energy Forecasting and Coordination Prototype

## Features
- Demand Forecasting
- Peak Load Detection
- State-Based Optimization Engine
- Explainable AI Decisions
- Streamlit Dashboard

## Run Locally

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

Architecture

Data → Forecast → State Detection → Optimization → Dashboard
